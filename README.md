# Sistema de Gerenciamento de Produtos e Cupons

API robusta para gestão de produtos, descontos e cupons promocionais com validações avançadas e controle transacional.

## ⚙️ Configuração

```bash
cd frontend
cp .env.example .env
npm install

# Modo desenvolvimento:
npm run dev

# Modo produção:
npm run build
npm run start
```
